# GovindTech App

A full-stack web application deployed on AWS infrastructure.

## Tech Stack
- Node.js + Express
- MySQL (AWS RDS)
- Nginx (Reverse Proxy)
- Docker
- AWS EC2 (Private Subnet via Bastion)
- GitHub Actions (CI/CD)
- AWS CloudWatch (Monitoring)

## Deployment
Automatically deployed via GitHub Actions on every push to `main` branch.
